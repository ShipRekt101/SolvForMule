Vue d'ensemble-
Ceci est un outil sur votre calculatrice TI-84 qui résout les problèmes de mathématiques pour vous. Par exemple, si vous vouliez trouver la surface
d'un Prisim Rectangulaire, au lieu d'utiliser toute la formule et de la taper, vous pouvez aller au programme et taper les mesures
comme la longueur, la largeur et la hauteur, et il vous donnera la réponse. J'ai fourni des captures d'écran des programmes en action et vous pouvez
Vérifie-les.
//
//
//
Programmes inclus:
-Coordinate Midpoint et Distance finder; CDGEOFOR dans la calculatrice
Zone du cercle CIRCAREA dans la calculatrice
-Circonférence; CIRCUMFR dans la calculatrice
-Einear Equation Checker; EQUCHECK dans la calculatrice
-Pythagorean Theorum; PYGNTHRM dans la calculatrice
-Volume finder d'un cylindre, pyramide, cône et sphère; SHAPEPCK dans la calculatrice
-Simple intérêt Finder; SIMPINTR dans la calculatrice
-Surface surface formule pour n'importe quel prisme; SURFAREA dans la calculatrice
-Triangle zone; TRIAREA dans la calculatrice
-Volume de tout prisme; VOLMRECT dans la calculatrice
-Volume d'une sphère; VOLSPHRE dans la calculatrice
//
//
//

Étapes à suivre pour installer
Pour installer SolvForMule:
1. Vous devez dissocier le groupe appelé "Formula Programs", (il apparaîtra comme FRMPRGMS)
2. Ensuite, sélectionnez le programme que vous souhaitez exécuter
// * Les programmes inclus seront sous des noms différents et ils sont listés ci-dessus dans la section des programmes inclus pour votre
commodité.
3. Profitez!
